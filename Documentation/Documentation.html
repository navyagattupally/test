Logo

To change the logo, replace the "logo.png" in the images folder
Default logo size 108x52 px, as you scroll down the page, the navbar shrinks and logo becomes smaller.
.navbar gets .tiny class when you scroll down. You should change the default width values to match your logos width.

NAVBAR DEFAULT

.navbar-brand img { 
    width:108px;
  }

NAVBAR SHRINKED

.navbar-tiny .navbar-brand img { 
    width:86px;
  }
